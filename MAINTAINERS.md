# Maintainers

## Primary maintainer

- Sebastian Legarraga ([@slegarraga](https://github.com/slegarraga))

## Maintainer responsibilities

- Review issues and pull requests.
- Maintain releases and npm package publishing.
- Keep reasoning-output fixtures current for LLM JSON extraction.
- Triage compatibility, security and documentation reports.

## Scope

`json-from-llm` is maintained as a public MIT package for extracting JSON from
LLM responses wrapped in reasoning tags, markdown fences or prose. The primary
maintainer controls the GitHub repository, npm package, release tags and
security policy.
